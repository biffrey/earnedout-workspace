# Broker Outreach Templates

## Template Selection Logic

1. **Aviation leads (Part 135, Part 145):** Use Template C (Aviation-specific)
2. **Price-drop re-outreach:** Use Template D (Price-drop follow-up)
3. **All others:** Use Template A (Default)
4. **A/B testing:** Rotate subject line variants only — body text stays consistent

---

## A/B Testing Rules

- **What rotates:** Subject line only (Variant 1 vs Variant 2)
- **Selection method:** Odd listing IDs → Variant 1, Even → Variant 2
- **Body text:** Stays consistent within each template — response-rate variance comes primarily from subject lines
- **Tracking:** Log which variant was used in the Airtable Notes field
